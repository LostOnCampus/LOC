# LostOnCampus 실행 가이드

## 1. DB 접속 정보 설정

`config/db.php`에서 아래 값을 실제 Cloud DB for MySQL 정보로 수정한다.

```php
define('DB_HOST', getenv('LOC_DB_HOST') ?: 'db-47u9u6.vpc-cdb.ntruss.com');
define('DB_PORT', getenv('LOC_DB_PORT') ?: '3306');
define('DB_NAME', getenv('LOC_DB_NAME') ?: 'lost_on_campus');
define('DB_USER', getenv('LOC_DB_USER') ?: 'user');
define('DB_PASS', getenv('LOC_DB_PASS') ?: 'DB비밀번호');
```

서버 환경변수로 관리하려면 Apache 설정 또는 쉘 환경에 다음 값을 지정한다.

```bash
export LOC_DB_HOST="db-47u9u6.vpc-cdb.ntruss.com"
export LOC_DB_PORT="3306"
export LOC_DB_NAME="lost_on_campus"
export LOC_DB_USER="DB계정명"
export LOC_DB_PASS="DB비밀번호"
```

## 2. 테이블 생성 확인

DB 서버에 접속한다.

```bash
mysql -h db-47u9u6.vpc-cdb.ntruss.com -P 3306 -u DB계정명 -p lost_on_campus
```

테이블이 없으면 `sql/schema.sql`의 내용을 실행한다.

```sql
SOURCE /root/web_project/sql/schema.sql;
```

## 3. Apache 웹 루트 배치

Ubuntu Apache 기본 웹 루트는 `/var/www/html`이다. 프로젝트 파일을 웹 루트 기준으로 배치한 뒤 권한을 맞춘다.

```bash
sudo rsync -a /root/web_project/index.php /root/web_project/config /root/web_project/includes /root/web_project/css /root/web_project/js /root/web_project/pages /root/web_project/api /var/www/html/
sudo chown -R www-data:www-data /var/www/html
sudo find /var/www/html -type d -exec chmod 755 {} \;
sudo find /var/www/html -type f -exec chmod 644 {} \;
sudo systemctl restart apache2
```

## 4. 접속 확인

브라우저에서 아래 주소로 접속한다.

```text
http://서버공인IP
```

확인할 기능:

- 메인 페이지 표시
- 분실물/습득물 등록
- 목록 조회
- 상세 조회
- 검색
- 수정
- 삭제
- 문의/제보 저장

## 5. 문제 확인 명령

Apache 상태:

```bash
sudo systemctl status apache2 --no-pager
```

PHP 모듈:

```bash
php -v
php -m | grep -i mysql
php -m | grep -i pdo
```

DB 접속:

```bash
mysql -h db-47u9u6.vpc-cdb.ntruss.com -P 3306 -u DB계정명 -p lost_on_campus
```

## 6. 검증 및 오류 처리 테스트

브라우저에서 아래 항목을 확인한다.

| 테스트 | 기대 결과 |
|---|---|
| 게시글 등록 화면에서 필수값을 비우고 제출 | 오류 메시지 목록이 표시되고 입력값이 보존된다. |
| 제목, 장소, 연락처를 최대 길이보다 길게 제출 | 길이 제한 오류가 표시된다. |
| 게시글 수정 시 틀린 비밀번호 입력 | 수정되지 않고 비밀번호 오류가 표시된다. |
| 게시글 삭제 시 틀린 비밀번호 입력 | 삭제되지 않고 비밀번호 오류가 표시된다. |
| 존재하지 않는 게시글 ID 접근 | PHP 오류 원문 대신 게시글 없음 안내가 표시된다. |
| 문의/제보 필수값 누락 | 저장되지 않고 오류 메시지 목록이 표시된다. |
| 검색어 없이 검색 화면 접근 | 검색어 입력 안내가 표시된다. |
| DB 접속 정보 임시 오류 | DB 비밀번호나 서버 경로가 노출되지 않고 일반 오류 안내가 표시된다. |
